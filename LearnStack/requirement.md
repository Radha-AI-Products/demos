# LearnStack UI Prompts

## 1. Dashboard

**Prompt:**

Design a learning platform UI called "LearnStack" using a light theme with a clean, modern, minimal style. Use rounded corners, soft shadows, subtle borders, and consistent spacing.

Create a full layout with:

A sidebar with navigation items:
- Dashboard
- My Courses
- Learn
- Explore
- Assignments

A top header with a search bar, notification icon, and user avatar.

Generate the Dashboard page as the main content area with the following sections:
- Continue Learning (featured card with course progress)
- My Courses (small grid of course cards)
- Learning Activity (chart or graph placeholder)
- Explore Courses (horizontal scroll cards)
- Assignments (list of upcoming tasks)

Ensure all sections are visually distinct using cards and proper spacing.

---

## 2. My Courses

**Prompt:**

Design the "My Courses" page for the LearnStack platform using a light theme with a clean, modern, minimal UI style. Maintain the same sidebar and header layout.

The page should display:
- A grid of enrolled course cards
- Each card should include:
  - Course thumbnail
  - Course title
  - Instructor name
  - Progress bar with percentage
- Include at least 6–8 courses

Add:
- A filter or sorting option at the top
- A "Continue Learning" quick action on hover or within the card

Ensure consistent spacing, rounded corners, and soft shadows.

---

## 3. Learn

**Prompt:**

Design the "Learn" page for the LearnStack platform.

Create a course learning interface with:

A main content area containing:
- Video player or lesson content viewer
- Lesson title and description
- Tabs for Notes, Discussion, and Resources

A right or left sidebar showing:
- Course modules and lessons
- Highlight the current lesson
- Completed lessons marked clearly

Include:
- Previous and Next lesson navigation
- A progress indicator for course completion

Keep the UI clean, focused, and distraction-free.

---

## 4. Explore

**Prompt:**

Design the "Explore" page for the LearnStack platform.

The page should include:
- A top search bar
- Filters for category, difficulty level, and duration

Display a grid of course cards with:
- Thumbnail
- Title
- Instructor
- Rating
- Level (Beginner, Intermediate, Advanced)

Include:
- A "Trending Courses" or "Featured" section
- Category tags for quick browsing

Use a visually rich but clean layout with proper spacing and card-based design.

---

## 5. Assignments

**Prompt:**

Design the "Assignments" page for the LearnStack platform.

Display assignments in a structured list or table format with:
- Assignment title
- Course name
- Due date
- Status (Pending, Submitted, Overdue)

Include:
- Filters for status and due date
- Visual indicators for urgency (colors or icons)
- Actions like "View" or "Submit"

Ensure rows or cards have subtle borders, rounded corners, and clean spacing.
